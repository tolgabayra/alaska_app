## Installation

Alaska requires [Node.js](https://nodejs.org/) v16+ to run.

Install the dependencies and devDependencies and start the server.

### Dont forget the env variables

### Account-Service
```sh
cd account-service
npm i
npm start
```


### Customer-Service
```sh
cd account-service
npm i
npm start
```


### Notification-Service
```sh
cd account-service
npm i
npm start
```




## Docker
